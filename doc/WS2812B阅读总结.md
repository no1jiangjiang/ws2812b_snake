# WS2812B

## 1、特点

1、IC控制电路与LED点光源公用同一电源

2、内置上电复位、掉电复位

**3、每个像素点的三基色可以实现256级的亮度显示、能够完成1677万种颜色的全真显示，扫描频率为2kHZ**

**4、串行级联接口，能够通过一根数据线完成数据的接收、解码(我们实验操作的就是如此)**

5、在刷新率为30帧时，级联数不小于1024点

**6、数据发送速度达到800kbps**

## 2、引脚及传输特性

### 1、引脚功能

| 序号 | 符号 | 管脚名   | 功能描述     |
| ---- | ---- | -------- | ------------ |
| 1    | V+   | 电源     | 5V供电       |
| 2    | V-   | 地       | 电源接地     |
| 3    | IN   | 数据输入 | 控制数据输入 |
